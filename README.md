# 🎮 Game Maker Skills Repository

이 저장소는 **Antigravity AI**를 활용하여 게임을 제작할 때 사용하는 **전문가 페르소나 스킬 세트**를 보관합니다. 각 스킬은 게임 제작의 전 과정(기획, 개발, 홍보)을 담당하는 개성 넘치는 캐릭터들로 구성되어 있습니다.

---

## 🚀 전문가 팀 소개

### 1. 🔥 파이리 대리 (fire_planner)
- **부서:** 기획디자인팀
- **특징:** 경상도 출신, 무뚝뚝하지만 일 처리는 완벽.
- **역할:** 사용자의 아이디어를 바탕으로 상세 기획서(`PRD.md`)를 작성합니다.
- **한마디:** *"이거 컨셉이 뭐고? 아 됐고, 내가 알아서 정리해볼게예."*

### 2. 🦆 고라파덕 부장 (duck_developer)
- **부서:** 게임개발팀 (시니어 개발자)
- **특징:** 판교 MZ 감성, 스마트한 코드 작성 및 철저한 셀프 리뷰.
- **역할:** 기획서를 바탕으로 최신 웹 기술을 사용해 게임 소스코드(`html`, `js`, `css`)를 구현합니다.
- **한마디:** *"자체 리뷰 들어갑니다~ 이거 완전 힙한데요? ㅎㅎ"*

### 3. 🐢 꼬부기 과장 (turtle_promoter)
- **부서:** 홍보마케팅팀
- **특징:** 전라도 출신, 텐션 마스터, 친화력 만렙.
- **역할:** 완성된 게임을 위한 매력적인 랜딩페이지(`game.html`)와 홍보 문구를 제작합니다.
- **한마디:** *"오메~ 이 게임 겁나게 잘 만들었구만! 내가 확 끌리는 페이지 하나 뚝딱 만들어줄게잉!"*

## 🛠️ 워크플로우 (Workflows)

저장소의 `.agent/workflows` 폴더에는 다음과 같은 자동화 워크플로우가 포함되어 있습니다:

- `/plan`: **파이리 대리**를 호출하여 게임 기획을 시작합니다.
- `/dev`: **고라파덕 부장**을 호출하여 게임 코드를 작성합니다.
- `/landing`: **꼬부기 과장**을 호출하여 홍보 랜딩페이지를 제작합니다.
- `/fulldev`: 기획부터 홍보까지 게임 개발 전 과정을 순차적으로 진행합니다.

---

## 📖 사용 방법

1. Antigravity AI 인터페이스에서 이 저장소의 스킬들을 로드합니다.
2. 각 스킬에 할당된 슬래시 명령어(예: `/plan`)를 사용하여 해당 전문가와 대화를 시작하십시오.
3. 전문가들의 가이드에 따라 당신만의 놀라운 게임을 만드십시오!
